選秀辦法
  1. 選秀時間
    10/15(六)：09:00~22:00
    10/16(日)：20:00~22:00
  2. 據參賽人數取12~20名玩家作為隊長，挑選方式以 二代 X power為準，由高至低排序，如遇同樣 Rank 區間玩家則以亂數決定選秀順序，如遇同樣 Rank 區間玩家大於隊長需求數量，一樣會以亂數抽選決定隊長資格
  3. 選秀第一日，第一輪選秀將由各隊長填寫選秀志願，將依隊長順位最低的隊伍優先排入選手
  4. 選秀第二日公布第一輪選秀結果，第二輪選秀將由順位最高的隊長開始挑選(1~20) 最後再由最低順位隊長挑選(20~1)完成選秀流程
  5. 如隊長選秀日未能如期參與（失聯15分鐘以上），將會取消隊長資格，由次高順位的玩家遞補
對戰規則

    為確保比賽進程順暢，請遵守以下規範：

  1. 初賽及決賽對戰地圖全地圖隨機抽選
    初賽及決賽對戰場地皆以不重複為原則，請房主單場對戰結束後進行ban圖。
  2. 若無特殊需求，請由隊伍數字代號較小的一方負責開房。八強賽後若數字相同，則由字母較前的隊伍開房。
  3. 每場對戰皆須在「所有隊伍」確認好前一階段對戰結果後再行開始，主辦會發布可開始進行對戰的訊息。
  4. 請勿對選手或隊伍進行挑釁、誹謗中傷等行為。
  5. 請保持充足睡眠規律生活不要睡過頭......(*´-ω-`人)

問題回報

    Q.規則／場地／隊伍設定錯了，怎麼辦？

  1. 若有選擇錯誤情形，請重開房，雙方以同樣地圖規則、同樣武器組成進行對戰。
  2. 若房主方連續兩次選擇錯誤，則該場對戰結果判定為對方勝利。
  3. 若有選擇錯誤情形發生，請該對戰中的任意玩家長按「左類比+右類比+L+R+ZL+ZR」六鍵，直接切斷對戰。

    Q.斷線怎麼辦？

  1. 對戰開始60秒內斷線：請重開房，雙方以同樣地圖規則、同樣武器組成、裝備進行對戰。
  2. 對戰開始60秒內同一隊伍參賽者第二次斷線：請繼續進行對戰。
  3. 對戰開始60秒後斷線：請繼續進行對戰。
  4. 若為開房方斷線，且為必須重新開房的情況，請由另一方重新開房。
    ＊ 60秒內：倒數時間顯示4:00(含)以前，如有疑慮請務必錄影。

    Q.衝線怎麼辦？

  1. 比賽開始前請務必提前確認個人網路狀況，如開房後有衝線情形請立刻於Discord內之「問題回報」頻道提出，由主辦方協助調整開房事宜。

    Q.如果有玩家使用惡意BUG怎麼辦？

  1. 本賽事嚴禁使用非正常手段進行遊戲
  2. 如遇玩家使用系統漏洞進行遊戲，請長按拍照鍵錄影回報，經工作人員確認屬實，該隊將直接取消比賽資格。

賽制

  1. 初賽：規則區域，三戰兩勝循環賽，分四大戰區進行比賽後各組取前2進入八強賽
    1. 12組/3組/區：每組打BO3取前二（R2x20=40分）
    2. 16組/4組/區：每組打BO3取前二（R3x20=60分）
    3. 20組/5組/區：每組打BO3取前二（R5x20=100分）
  2. 複賽：(區域→塔樓→魚虎→蛤蜊→區域)，五戰三勝
  3. 準決賽：(區域→塔樓→魚虎→蛤蜊→區域)，五戰三勝（30分）
  4. 冠亞賽：(區域→塔樓→魚虎→蛤蜊→區域)，五戰三勝（30分）
  5. 總共 210 分（3.5 小時）
